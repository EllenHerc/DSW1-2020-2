Roteiro e tecnlogias utilizadas:
- Banco de dados utilizado: MySql. 
- Certifique-se de ajustar, caso necessário, as configurações de conexão no arquivo "application.properties":

# MYSQL
spring.datasource.url = jdbc:mysql://localhost:3306/compravendaimoveist2?createDatabaseIfNotExist=true&useTimezone=true&serverTimezone=UTC
spring.datasource.username = root
spring.datasource.password = user

#localhost port
server.port = 8090 

- O BD é criado automaticamente pela aplicação, caso queira executá-lo manualmente rodar o script sql (db) que esta dentro da pasta BancoDeDadosT2;
- Utilize o comando "mvn spring-boot:run"; ou similar, para iniciar o projeto.
